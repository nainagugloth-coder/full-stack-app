# full-stack-app
# Student Feedback App

Full Stack Application

Frontend: React  
Backend: Python Flask  
Database: MongoDB (NoSQL)

## How to Run Backend

cd backend  
pip install -r requirements.txt  
python app.py  

## How to Run Frontend

cd frontend  
npm install  
npm start  

App runs on:
http://localhost:3000

Backend runs on:
http://localhost:5000
